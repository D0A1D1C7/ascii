ascii
=====

An image viewer for your terminal


![ascii running inside urxvt](https://raw.github.com/joshdk/ascii/master/img/screenshot.png "ascii running inside urxvt")


Setup
-----

### Install
    # make install


### Uninstall
    # make uninstall


Running
-------

    $ ascii IMAGE [CHARSET]

    $ ascii picture.jpg

    $ ascii picture.jpg " -+=$#"


Notes
-----

### Dependencies
*   Python2
*   PIL
